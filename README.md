# CPP_07

Le but est d’apprendre à utiliser les **templates** en C++, afin d’écrire du code générique et réutilisable.

## Compilation

```bash
make
```

## Contenu

* Introduction aux **templates de fonctions et de classes**
* Manipulation de types génériques sans duplication de code
* Mise en pratique avec des **exemples simples de containers et d’opérations**
